Description

E4X (Lite)

This is a E4X library making it easy to parse and manipulate XML documents. It is not a compliant implementaion of E4X, but it is very close to being so from the end-user’s point-of-view.

This library uses REXML as a backend. An option to use libxml may be added in the future for speed.
